## TL;DR

<div class="tldr">

This deck covers one thing: **how the Hermes agent is architected**.

**The seven components**

- 🧠 **Brain** (config) — replaceable AI model
- 👻 **Soul** (soul.md) — persona, tone, taboos
- 💾 **Memory** (memory.md + user.md) — compressed notebook + vector
- 📖 **Manual** (skill.md) — self-written procedures
- 🛠 **Hands** (tools) — terminal, file, web, delegate
- 📡 **Channel** — 20+ gateway (Discord, Telegram, Slack…)
- ⏰ **Alarm** (cron) — auto-wake, work, sleep

**Three principles**

1. Cost zero (local) · 2. Best performance (cloud) · 3. Zero data contamination
